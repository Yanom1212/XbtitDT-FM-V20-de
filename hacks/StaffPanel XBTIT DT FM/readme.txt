Staff Panel (extended version) v.1.0 by Lupin
---------------------------------------------

- upload the folder into your hack's folder
- go to your admincp, hacks settings, install "staff Panel"
- in order to allow your staff to access some part of the admincp you need:
  * in users group settings, allow the groups you want to access admincp
  * in Staff Panel settings, select the level you want to edit, click Select
  * check the part you want to allow or uncheck the part you don't want, click Save
  * if you have need to allow access on custom parts, goto to "add new section", compile the field and click confirm
    the fields are explain in the form, if you have doubt ask me.
